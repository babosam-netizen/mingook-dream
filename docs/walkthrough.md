# 워크스루: 학급별 나라이름 개인화 및 여정(Journey) 체계 정착 (v0.9.91)

이번 업데이트에서는 학생들이 수업에 더욱 몰입할 수 있도록 학급 고유의 '나라 이름'을 시스템 전반에 반영하고, 'Phase'라는 기술적 용어를 '여정'이라는 교육적 용어로 완전히 대체했습니다.

## 1. 주요 변경 사항

### 🏢 학급 고유 나라이름 설정 도입
교사용 관리 화면의 [학급 설정] 메뉴에서 우리 반만의 특별한 나라 이름을 설정할 수 있습니다. 설정된 이름은 즉시 저장되어 모든 학생 화면에 반영됩니다.

- **설정 위치**: 관리 화면 > 학급 설정 > 나라이름 입력 필드
- **자동 폴백**: 나라 이름이 설정되지 않은 경우 기존처럼 '우리 반' 또는 '우리 동네'로 자연스럽게 표시됩니다.

### 🔄 다이내믹 텍스트 치환 (Dynamic Replacement)
시스템 곳곳에 산재해 있던 '우리 동네', '우리 반' 표현을 설정된 나라 이름으로 자동 치환합니다.

- **워크플로 가이드**: 단계별 미션 안내 문구 내 키워드 치환
- **여론조사**: 질문 및 선택지 내 키워드 치환 (학생들의 현장감 증대)
- **예외 처리**: 첫 번째 여정 2번 활동(**우리 동네 문제 발굴**)은 교육적 의도를 고려하여 '우리 동네' 표현을 명시적으로 유지합니다.

### 🗺️ 여정(Journey) 및 우리말 용어 정착
전체적인 용어 체계를 순화하여 초등학생 눈높이에 맞춘 교육 환경을 완성했습니다.

- **Phase → 여정**: '첫 번째 여정', '두 번째 여정' 등 순수 우리말 표현 사용
- **핵심 화면 개인화**: 
  - Phase 1: 최우선 과제 잠금 전광판에 나라 이름 노출
  - Phase 3: '최우선 과제 보고서' 발행 주체로 나라 이름 국정운영위원회 명시

## 2. 주요 화면 변경 사항

### 교사 관리 화면
- **학급 정보 요약**: 현재 진행 중인 여정(Phase) 정보와 설정된 나라 이름을 한눈에 확인 가능
- **학급 설정**: 나라 이름을 자유롭게 편집할 수 있는 입력 폼 추가

### 학생 화면
- **여론조사 피드**: 질문 문구 내에 우리 반의 나라 이름이 자연스럽게 포함되어 노출
- **활동 가이드**: 단계별 미션 내용이 우리 반만의 고유한 서사로 변환되어 전달

## 3. 검증 결과
- **동적 치환**: 나라 이름 변경 시 새로고침 없이(또는 즉시) UI에 반영됨을 확인
- **예외 로직**: Phase 1 Step 2에서 '우리 동네' 표현이 정상적으로 보존됨을 확인
- **빌드 및 배포**: v0.9.91 버전으로 최종 빌드하여 NAS 배포 완료
